# Média de Valores Positivos em Java

Programa em Java que lê 6 números informados pelo usuário, conta quantos são positivos e calcula a média desses valores.

## Conceitos usados
- Scanner
- Estrutura de repetição `for`
- Condicional `if`
- Cálculo de média

## Saída
- Quantidade de valores positivos
- Média dos valores positivos com uma casa decimal
